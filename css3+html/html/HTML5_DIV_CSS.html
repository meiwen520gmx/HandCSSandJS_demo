学习八大行星

  • 金星
  • 自转方向：自东向西
  • 太阳系中第六大行星，中国古代称之为太白或太白金星
  • 木星
  • 自转方向：自西向东
  • 木星是离太阳第五颗行星，亦为太阳系行星中质量最大的一颗
  • 地球
  • 轨道半径：149,600,000千米
  • 地球是距太阳第三颗，也是太阳系第五大行星

关于行星

  • 早在公元前3000年的苏美尔时代，人们便发现了水星，古希腊人赋于它两个名字：当它初现于清晨时称为阿波罗，当它闪烁于夜空时称为赫耳墨斯
  • 柯伊伯带是处于海王星轨道以外的一个太空区域，在这个区域里，到处是冰冷、岩石状的天体

关于发现行星的人

  • 戈达德

    作为马里兰州绿带市美国宇航局（NASA）戈达德空间飞行中心的一名行星科学家，Glaze多年来一直力推重返这颗距离地球最近的邻居

  • 郑永春

    现在大家对月球探测和火星探测都耳熟能详，嫦娥系列月球探测器可谓是“网红”一族。但是十六年前，月球研究可谓个大冷门，甚至比现在北大的古生物专业更冷，几乎没有人愿意学跟月球相关的专业
